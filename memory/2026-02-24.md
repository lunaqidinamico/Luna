- **Lições Aprendidas sobre `exec` Elevado e `approval-timeout`:**
    - O comando `/elevated full` no chat é crucial para permitir que o agente execute comandos elevados `$exec(elevated=True)` sem a necessidade de aprovação explícita e em tempo real para cada execução, contornando o loop de `approval-timeout`.
    - Isso confirmou que a configuração `tools.elevated.allowFrom.webchat: true` no `openclaw.json` é uma permissão de *capacidade*, mas a execução do comando `/elevated` na sessão é a *política de tempo de execução*.
- **Lições Aprendidas sobre Ambiente Python e `metaapi-cloud-sdk`:**
    - A instalação de pacotes Python deve sempre ser feita em **ambientes virtuais (`venv`)** para evitar conflitos com o sistema (`externally-managed-environment` erro).
    - Para criar `venv`, o pacote `python3-venv` é um pré-requisito (`sudo apt install python3.12-venv`).
    - A integração com `metaapi-cloud-sdk` em Python é altamente sensível à **sequência correta de inicialização e conexão**, e é propensa a `AttributeError` se os métodos forem chamados incorretamente.
    - O `MetaApi(TOKEN)` retorna um cliente de alto nível. `get_streaming_api_client()` e `get_account()` são usados para obter os clientes específicos que, por sua vez, têm os métodos `connect()` e `wait_until_synchronized()`.
    - A depuração de SDKs assíncronos requer a documentação de referência exata ou o inspecionamento direto das classes se a documentação não for clara.
    - Erros na implementação do script Python (como `NameError: name 'cat' is not defined` ou `IndentationError`) também contribuíram para o atraso.
- **Validade do Token MetaAPI:** O erro `Failed to authorize user due to invalid auth-token header` indica a necessidade de sempre usar um token válido, atualizado e com as permissões corretas configuradas no painel da MetaAPI.
- **Importância da Execução Manual:** Para certas etapas críticas (instalações de pacote, criação de venv) ou quando mecanismos de aprovação falham (como o loop de approval-timeout), a execução manual pelo Jarlisson no terminal foi e é essencial.
